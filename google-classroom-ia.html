Google Classroom + IA

Como usar IA dentro do Google Classroom para ganhar tempo e melhorar as aulas

Como integrar IA no Classroom

  • Crie tarefas com Gemini: "Gere 10 questões sobre frações"
  • Use Copilot para corrigir redações: cole o texto e peça feedback
  • Crie rubricas com IA: "Crie rubrica para redação ENEM"
  • Personalize atividades: "Adapte esta atividade para aluno com TDAH"

Dicas práticas

  • Compartilhe prompts prontos com alunos (ex.: "Use Gemini para estudar")
  • Ensine ética: "IA é ferramenta, não cola"
  • Use Classroom para feedback individual rápido
